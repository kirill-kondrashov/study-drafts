# Study Drafts

A collection of hand-written notes compiled while following mathematics and
computer-science lecture courses on YouTube. Each subfolder corresponds to a
single course: it holds the scanned notes (per-lecture PDFs plus a
`compiled.pdf`) and a README linking to the original YouTube playlist.
